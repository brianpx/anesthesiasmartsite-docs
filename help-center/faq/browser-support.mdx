## Which browsers are supported?

Anesthesia SmartSite supports current versions of:

| Browser | Minimum Version |
|---------|----------------|
| Google Chrome | Latest 2 major versions |
| Mozilla Firefox | Latest 2 major versions |
| Safari | Latest 2 major versions |
| Microsoft Edge | Latest 2 major versions |

<Callout kind="info">
  For the best experience, use the latest version of your browser. Older versions may not support all platform features.
</Callout>

## Does the platform work on mobile devices?

Yes. The platform is responsive and optimized for mobile browsers on iOS and Android devices. All content — protocols, quizzes, presentations, and account management — is accessible on mobile.

## Is there a mobile app?

Currently, Anesthesia SmartSite is a web-based platform accessed through your mobile browser. There is no dedicated native app at this time.

## Does the platform work offline?

No. An internet connection is required to access platform content, take quizzes, and track CME progress.

## Are accessibility features available?

The platform follows web accessibility standards including:
- Keyboard navigation support
- Screen reader compatibility
- Adjustable text sizes through browser settings
- Dark mode for reduced eye strain

If you encounter accessibility issues, report them to [support@anesthesiasmartsite.com](mailto:support@anesthesiasmartsite.com).

## The site is not loading properly. What should I try?

See [Page Won't Load](/help-center/troubleshooting/page-load-error) for troubleshooting steps, including clearing cache, disabling extensions, and checking institutional firewall settings.
